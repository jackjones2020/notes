# Agent 系统的两个前提：Trust by Default 和 Egalitarian

## 核心观点

Agent 系统要发挥最大价值，需要两个前提：

### 1. Trust by Default（默认信任）
- 默认信任 Agent 的能力和输出
- 不预设 Agent 不可靠，而是先信任再验证
- 与传统的"层层审批、最小权限"思路相反

### 2. Egalitarian（平等主义）
- AI 能力必须向**每个人**开放，不分层级、部门
- **所有 Agent 对话默认对全员可见**
- 只有这样，Agent 才能拿到足够广的上下文

## 与企业文化的冲突

这与大多数 `command and control`（命令与控制）式的企业是相反的——传统企业强调：
- 信息按需分配（need-to-know）
- 审批流程控制访问
- 层级壁垒

但在 Agent 时代，**上下文就是燃料**。信息的广度决定了 Agent 能做什么级别的决策。
